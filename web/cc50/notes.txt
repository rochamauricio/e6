- float resposta = 5 / 2; 			# realiza divisão inteira: resposta será 2.000000
- float resposta = 5.0 / 2; 		# resposta será 2.500000 
- float resposta = 5 / 2.0; 		# resposta será 2.500000 
- float resposta = (float) 5 / 2; 	# resposta será 2.500000 

- ver flags de compilação para forçar que não compile quando existir um warning - bom para evoluir nossas habilidades
- ver estilos (verde, vermelho e amarelo) no forum do cs50: https://cs50.readthedocs.io/style50/
- typecasting converter char em int para criptografia:	char c = 'a'; int x = (int) c;
