# Atalhos

## Abre nova janela
	ctrl + shift + n

## Abre arquivo:
	ctrl + o

	ctrl + k ctrl + o

## Abrir janela de arquivos/comandos:
	ctrk + p
	digite > para executar comandos

## Abrir / fechar um terminal:
	ctrl + shift + '
